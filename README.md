# Computer Architecture Course Repository

## Overview

This repository contains educational materials for the **Computer Architecture** course offered at the **Federal University of Rio Grande do Norte (UFRN)**, within the **Department of Computing and Technology (DCT)** at **CERES**.

The repository provides structured access to lecture slides, video lessons, and supporting materials designed to support undergraduate students in understanding the fundamental and advanced concepts of computer architecture.

## Course Objectives

### General Objectives

* To understand the hardware components of a computer system.
* To analyze the operation of the various modules that compose a computing system.
* To develop a critical perspective regarding performance requirements in computer systems.

### Specific Objectives

* To understand fundamental mechanisms such as:

  * Communication between system modules
  * Memory storage organization
  * Instruction sequencing
  * Interrupt handling and arithmetic operations
* To study the main hardware structures of computer systems.
* To evaluate the impact of architectural decisions on system performance across different applications.

---

## Course Content

The course covers the following topics:

1. Basic computer structure
2. Central Processing Unit (CPU)
3. Bus structures
4. Instruction Set Architecture (ISA)
5. CPU organization and operation
6. Pipeline and instruction-level parallelism
7. Memory hierarchy and organization
8. Input/Output (I/O) systems
9. Operating system support
10. Architectural standards (RISC vs CISC)
11. Multiprogramming
12. Processor-level parallelism
13. Dedicated architectures (DSPs, FPGAs, AI accelerators)
